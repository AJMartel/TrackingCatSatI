# Tracking CatSat I

Este es un sistema de para el segimiento en estacion terrena del CatSat I en tiempo real por medio de GPS y una red de radio frecuencia.


## Tecnologias del pryecto.

- Node.js
- Socket.io
- Lora 915Mhz
- Stylus
- Jade
- CatSat
- Open Street Map
